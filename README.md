# react-native-market-helper

跳转应用商店的帮助类，对应系统跳转对应的应用商店

## Installation

```sh
npm install react-native-market-helper
```

## Usage

```js
import { open } from "react-native-market-helper";

// ...

const appStoreId = 1397766350;
open(appStoreId);
```
